Preguntas frecuentes

¿La landing ya puede conectarse a un backend real?

Si. El frontend queda preparado para usar una API configurable y el backend ya incluye un endpoint de contacto listo para validación y envío.

¿Se puede publicar aunque todavía no haya credenciales?

Si. Puedes publicar el frontend primero y dejar el formulario en modo demo hasta conectar la API cuando tengas hosting o acceso al servicio.

¿La estructura sirve para mostrar trabajos reales?

Si. Solo necesitas reemplazar textos de ejemplo, enlaces y métricas por tus datos definitivos.
